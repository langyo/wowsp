# WoWSP

**لوحة تحليل المعارك لعالم السفن — مراجعة إعادات التشغيل وتراكب داخل اللعبة**

WoWSP هو لوحة تحليل معارك من الجيل التالي لعالم السفن (World of Warships). يعمل في وضعين: مراجعة مستقلة لإعادات التشغيل (خريطة ثلاثية الأبعاد هولوغرافية) وتراكب شفاف داخل اللعبة.
