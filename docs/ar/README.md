# WoWSP

**World of WarShip Panel — مراجعة الإعادة & تراكب داخل اللعبة**

WoWSP هو لوحة تحليل معارك من الجيل التالي للعبة World of Warships.

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1rem;margin:2rem 0">

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem;direction:rtl;text-align:right">

**<span style="color:#7aa2f7">مراجعة الإعادة</span>**

افتح أي ملف `.wowsreplay` وشاهد المباراة كاملة على خريطة ثلاثية الأبعاد هولوغرافية — دون الحاجة لتشغيل اللعبة.

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem;direction:rtl;text-align:right">

**<span style="color:#7aa2f7">عارض السفن ثلاثي الأبعاد</span>**

تصفح جميع السفن كنماذج هولوغرافية منخفضة التفاصيل. تدوير، تكبير، وفحص تخطيط الدروع.

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem;direction:rtl;text-align:right">

**<span style="color:#7aa2f7">التراكب داخل اللعبة</span>**

يعرض تراكب شفاف تشكيلات الفرق مباشرة أثناء اللعب، يظهر فقط عند الضغط على Tab.

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem;direction:rtl;text-align:right">

**<span style="color:#7aa2f7">الإحصائيات والتحليلات</span>**

تعمق في تاريخ معاركك مع إحصائيات لكل سفينة، مواسم التصنيف، واتجاهات الأداء.

</div>

</div>
