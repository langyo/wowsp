# WoWSP

**戰艦世界戰況面板 — 回放復盤 & 遊戲內浮層**

WoWSP 是面向《戰艦世界》的新一代戰況分析儀表板。

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1rem;margin:2rem 0">

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">回放復盤</span>**

開啟任意 `.wowsreplay` 檔案，在 3D 全息地圖上回看整場對局——無需啟動遊戲。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">3D 艦船檢視器</span>**

瀏覽遊戲中所有艦船的低多邊形全息模型，旋轉、縮放、檢視裝甲佈局。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">遊戲內浮層</span>**

透明浮層在遊戲中即時顯示雙方隊伍資訊，僅在按住 Tab 時可見。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">數據洞察</span>**

深入分析你的戰鬥歷史，包括單船數據、排位賽季和表現趨勢。

</div>

</div>
