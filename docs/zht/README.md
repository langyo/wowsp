# WoWSP

**戰艦世界戰況面板 — 回放復盤 & 遊戲內浮層**

WoWSP 是面向《戰艦世界》的新一代戰況分析儀表板。

## 功能特性

### 回放復盤

開啟任意 `.wowsreplay` 檔案，在 3D 全息地圖上回看整場對局——無需啟動遊戲。

### 3D 艦船檢視器

瀏覽遊戲中所有艦船的低多邊形全息模型，旋轉、縮放、檢視裝甲佈局。

### 遊戲內浮層

透明浮層在遊戲中即時顯示雙方隊伍資訊，僅在按住 `Tab` 時可見。

### 數據洞察

深入分析你的戰鬥歷史，包括單船數據、排位賽季和表現趨勢。
