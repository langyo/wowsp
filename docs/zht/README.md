# WoWSP

**戰艦世界戰況分析儀表板 — 錄像回顧與遊戲內覆蓋層**

WoWSP 是面向《戰艦世界》的新一代戰況分析儀表板。它有兩種工作模式：獨立的錄像回顧（全息 3D 地圖）與遊戲內透明覆蓋層。
