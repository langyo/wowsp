# WoWSP

**World of Warships 戦況分析ダッシュボード — リプレイレビューとゲーム内オーバーレイ**

WoWSP は『World of Warships』向けの次世代戦況分析ダッシュボードです。2 つのモードで動作します。独立したリプレイレビュー（ホログラフィック 3D マップ）と、ゲーム内の透過オーバーレイです。
