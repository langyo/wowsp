# WoWSP

**World of WarShip Panel — リプレイレビュー & ゲーム内オーバーレイ**

WoWSP は『World of Warships』向けの次世代戦況分析ダッシュボードです。

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1rem;margin:2rem 0">

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">リプレイレビュー</span>**

`.wowsreplay` ファイルを開くだけで、ホログラフィック 3D マップで試合全体を再生——ゲームの起動は不要です。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">3D 艦船ビューア</span>**

全艦船をローポリのホログラフィックモデルで閲覧。回転、ズーム、装甲配置の確認が可能です。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">ゲーム内オーバーレイ</span>**

Tab キーを押している間だけ、透明なオーバーレイで両チームの情報をリアルタイム表示します。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">統計 & 分析</span>**

艦船ごとの統計、ランク戦シーズン、パフォーマンス推移など、戦績を詳細に分析できます。

</div>

</div>
