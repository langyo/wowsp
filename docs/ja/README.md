# WoWSP

**World of WarShip Panel — リプレイレビュー & ゲーム内オーバーレイ**

WoWSP は『World of Warships』向けの次世代戦況分析ダッシュボードです。

## 機能

### リプレイレビュー

`.wowsreplay` ファイルを開くだけで、ホログラフィック 3D マップで試合全体を再生——ゲームの起動は不要です。

### 3D 艦船ビューア

全艦船をローポリのホログラフィックモデルで閲覧。回転、ズーム、装甲配置の確認が可能です。

### ゲーム内オーバーレイ

`Tab` キーを押している間だけ、透明なオーバーレイで両チームの情報をリアルタイム表示します。

### 統計 & 分析

艦船ごとの統計、ランク戦シーズン、パフォーマンス推移など、戦績を詳細に分析できます。
