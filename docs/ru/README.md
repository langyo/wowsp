# WoWSP

**World of WarShip Panel — просмотр реплеев & внутриигровой оверлей**

WoWSP — это панель анализа боя нового поколения для World of Warships.

## Возможности

### Просмотр Реплеев

Откройте любой `.wowsreplay` и смотрите весь матч на голографической 3D-карте — запуск игры не требуется.

### 3D-Просмотр Кораблей

Просматривайте все корабли в виде low-poly голографических моделей. Вращение, масштабирование, осмотр брони.

### Внутриигровой Оверлей

Прозрачный оверлей показывает составы команд в реальном времени, видимый только при удержании `Tab`.

### Статистика

Глубокий анализ боевой истории: статистика по кораблям, ранговые сезоны и тенденции производительности.
