# WoWSP

**战舰世界战况面板 — 回放复盘 & 游戏内浮层**

WoWSP 是面向《战舰世界》的新一代战况分析仪表盘。

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1rem;margin:2rem 0">

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">回放复盘</span>**

打开任意 `.wowsreplay` 文件，在 3D 全息地图上回看整场对局——无需启动游戏。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">3D 舰船查看器</span>**

浏览游戏中所有舰船的低多边形全息模型，旋转、缩放、检视装甲布局。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">游戏内浮层</span>**

透明浮层在游戏中实时显示双方队伍信息，仅在按住 Tab 时可见。

</div>

<div style="background:#131e32;border:1px solid #1e3250;border-radius:1rem;padding:1.25rem">

**<span style="color:#7aa2f7">数据洞察</span>**

深入分析你的战斗历史，包括单船数据、排位赛季和表现趋势。

</div>

</div>
