# WoWSP

**战舰世界战况分析仪表盘 — 录像复盘与游戏内覆盖层**

WoWSP 是面向《战舰世界》的新一代战况分析仪表盘。它有两种工作模式：独立的录像复盘（全息 3D 地图）与游戏内透明覆盖层。
