# WoWSP

**战舰世界战况面板 — 回放复盘 & 游戏内浮层**

WoWSP 是面向《战舰世界》的新一代战况分析仪表盘。

## 功能特性

### 回放复盘

打开任意 `.wowsreplay` 文件，在 3D 全息地图上回看整场对局——无需启动游戏。

### 3D 舰船查看器

浏览游戏中所有舰船的低多边形全息模型，旋转、缩放、检视装甲布局。

### 游戏内浮层

透明浮层在游戏中实时显示双方队伍信息，仅在按住 `Tab` 时可见。

### 数据洞察

深入分析你的战斗历史，包括单船数据、排位赛季和表现趋势。
